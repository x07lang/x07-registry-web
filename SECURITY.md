# Security Policy

Thank you for helping to keep the X07 registry UI (x07.io) secure.

## Reporting a vulnerability

Please **do not** report security vulnerabilities via public GitHub issues, pull requests, or public discussions.

Preferred (if enabled): **GitHub Private Vulnerability Reporting**

1. Open this repository's **Security** tab.
2. Click **Advisories**.
3. Click **Report a vulnerability**.

Fallback: email **security@x07lang.org**.

When reporting, please include:

- A minimal reproduction (redact tokens and secrets).
- Affected URL(s) and browser details.
- Impact assessment.

## Supported versions

The hosted UI is deployed from the latest release; security fixes are applied by redeploying.

## Disclosure timeline

- **Acknowledgement:** within 72 hours.
- **Triage:** within 7 days.
- **Fix and deployment:** typically within 30 days.

## Credits

We will credit reporters in advisories and release notes unless you request otherwise.
