# Support

This repository contains the registry UI for https://x07.io/.

## Questions

If you have questions about publishing packages or using the toolchain, start with:

- X07 docs: https://x07lang.org/docs/packages/
- GitHub Discussions (recommended): https://github.com/x07lang/x07/discussions

## Bug reports

GitHub Issues in this repo are for actionable UI bugs.

Please include:

- Steps to reproduce
- Expected vs actual
- Browser + OS
- Network console output (redact tokens)

## Security issues

Do not open public issues for security problems.

See [SECURITY.md](SECURITY.md).
